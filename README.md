# Torshov Hometown Homepage 🌍

A hometown homepage project built with HTML and CSS as part of the Scrimba HTML & CSS course.

Unlike my previous tutorial-based project, this one was built independently as a solo practice challenge.

## What I Learned
- Building layouts using Flexbox
- Creating hero sections
- Structuring reusable content cards
- Improving spacing and alignment
- Working with background images
- Styling sections consistently

## Features
- Hero banner section
- Activity cards
- Local guide section
- Custom styling and color palette

## Tech Stack
- HTML5
- CSS3

## Challenges
- Aligning content properly with Flexbox
- Managing spacing between sections
- Keeping the design visually balanced
